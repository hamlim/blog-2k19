# Matt Hamlin's Blog

Welcome to my blog, this is an initial early version of the site, completly static and generated at
build time using tools like `MDX`, `prettier`, and `esm`.

### Posts:

<fileContext.Consumer>
  {({ files }) => (
    <ul>
      {files.map(file => (
        <li key={file}>
          <a href={file.replace('posts/', '').replace('.mdx', '.html')}>
            {file
              .replace('posts/', '')
              .replace('.mdx', '')
              .split('/')
              .reverse()[0]
              .split('-')
              .map(hunk => hunk[0].toUpperCase() + hunk.slice(1))
              .join(' ')}
          </a>
        </li>
      ))}
    </ul>
  )}
</fileContext.Consumer>
